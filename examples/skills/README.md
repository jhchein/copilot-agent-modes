# Example Skills

These are **reference examples** showing optional skills you can add under `.github/skills/`.

They are **not active** until copied into `.github/skills/`.

## How to use

1. Copy a skill directory from `examples/skills/` into `.github/skills/`.
2. Keep the `description` narrow so it activates only for relevant tasks.
3. Add prompt-level invocation guidance only where needed.

## Current examples

- `writing-quality/` — structured quality pass for outward-facing documentation.
